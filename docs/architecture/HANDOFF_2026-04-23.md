### The schematic / photographic / hybrid framing

The existing rendering is "schematic" — parametric, hand-modeled,
works for any venue with no scan required. Genjutsu adds
"photographic" — real-capture-based, true-to-life, requires a scan.
These coexist as render modes on a single scene graph (one Three.js
`Scene`, accessed via `editor-store.ts:scene: Scene | null`), not
as separate products.

The "hybrid" mode is a possible third option: schematic geometry
with photographic overlays in regions where panoramas exist (e.g.,
show the Grand Hall with photographic walls but a schematic
tablecloth that the user can recolor). Decision deferred.

This conceptual framing is one of the most important reframings
from the chat conversation and should anchor any work on
`docs/architecture/GENJUTSU.md`.
