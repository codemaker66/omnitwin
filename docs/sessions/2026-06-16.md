  unless `VENVIEWER_ALLOW_RUNTIME_QA_PUBLIC_EXPOSURE=true` is set.

  success.

  reports with payload identity, runtime-package preflight, signed-transform
  preflight, guardrail flags, and live readback row identity. Reports record no
  bearer token and refuse overwrite unless

  inspection mode via `RUNTIME_QA_INSPECT_REPORT_FILE`. Inspection exits before
  QA payload loading, token requirements, API preflight, or POST, and marks only

  `RUNTIME_QA_INSPECTION_FILE` writes a separate machine-readable inspection
  artifact and refuses overwrite unless

  are audit evidence, not authorization for another POST.

  `CAPTURE_CONTROL_INSPECTION_FILE=<path>` output so inspection decisions can
  be preserved as machine-readable evidence artifacts. Inspection artifacts

  identity, override state, blockers, and messages; overwrite requires
  `VENVIEWER_OVERWRITE_CAPTURE_CONTROL_INSPECTION=true`.
